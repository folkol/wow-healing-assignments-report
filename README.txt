# core priorities
- don't cap / waste maelstrom (dump with ES / EB when near cap or when MotE is active)
- LB on cd, but don't munch MotE
- EB is filler or to consume MotE if ES is on cd
- LB as filler

# cds
- stormkeeper on cd, ideally ~3s before pull or Ascendance
- Ascendance, align with stormkeeper and stack with trinktes/BL/pots
- Ancestral Swiftness / Nature's Swiftness: use for movement or high prio moments
